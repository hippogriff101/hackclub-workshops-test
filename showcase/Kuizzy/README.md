# Kuizzy

A Kahoot clone made using SweetAlert and WebSockets

## Running

Run

```bash
yarn
```

and then 

```bash
yarn start
```

and open

- `localhost:3000/host` - To be the host
- `localhost:3000` - To join the game as a player
